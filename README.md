# Globenet Portfolio Website

Production-grade portfolio website for **Globenet** — a web development agency in Awka, Anambra State, Nigeria.

**Tagline:** *Think It, We'll Build It...*

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build & Deploy

```bash
npm run build
npm start
```

Deploy to [Vercel](https://vercel.com) — zero extra configuration required.
